# commit.show

> Brand-friendly alias for the [`commitshow`](https://npmjs.com/package/commitshow)
> CLI so `npx commit.show audit …` works as well as `npx commitshow audit …`.

```bash
npx commit.show audit github.com/owner/repo
```

Same behaviour, same flags, same output as `commitshow`. Use whichever spelling
matches the way you (or your AI agent) think about the brand.

See the full docs at [npmjs.com/package/commitshow](https://npmjs.com/package/commitshow)
or [commit.show](https://commit.show).
